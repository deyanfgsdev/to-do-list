## Installation and setup

### Requirements:

- Node.js (>= 16.0)
- npm or yarn

### Instructions:

```
git clone https://github.com/deyanfgsdev/to-do-list.git
cd to-do-list
npm install
npm run dev
```

### E2E testing with Playwright:

```
# Run test
npx playwright test

# Run test in UI mode
npx playwright test --ui
```
